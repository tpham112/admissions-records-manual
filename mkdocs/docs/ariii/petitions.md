# Petitions (ARIII)



---

## N1 Drops (Academies)

**STAC:**
1. Detail into the desired section.
2. Insert in Status/Dt/Time/Reason: `D / Start Date of Class / 11:59 PM / N1`
3. Detail into Notes/Comments → Comments.
4. Timestamp.
5. Write comment (e.g., *"N1 Drop per Instructor and W. Reardon"*).
6. Save All.

---

## Student Change of Information Form

### Change Academic Program
**SPRO:**
1. Drill into current academic program → change status to **I** (Inactive) → Save.
2. Add new academic program:
    - Start Date: start of the term (`YYYYFA` or `YYYYSP`)
    - Input admit status.

### Change Home Campus
*(Check with Financial Aid first)*

**SPRO:**
1. Add end date to the old campus.
2. Add a new campus with a start date.

### Change Name
*(Student needs government-issued photo ID)*

**NAE:**
1. Edit name.
2. "Is this a name change?" → **Yes**
3. "Is this birth or other name change?" → **Other**

### Add/Change SSN
*(Student needs SSN card AND government photo ID)*

**NAE:** Change or add SSN.

### Change from CAP to Regular Student
**ASUM:** Check for active application and confirm student is regular and has HS diploma.

**SPRO:**
1. Insert a new line for student type → change from **CAP** to **RGLR**.
2. Add a new academic program.

## Probation/Dismissal Intervention

**PERC:**
1. Set end date to **yesterday**.
2. Detail into the hold → timestamp → leave comment: *"Counselor submitted a probation/dismissal intervention form."*

**SREP:**
3. If the student had a TTPRO, TTPR2, or TTPR3: add **Late Dates** for the student.
