### 📝 Processing Workflow by Request Type

#### 1. Verification Letters (Standard SAC Letter)
*Student needs a letter generated by the office.*
1. **Verify Payment:** Confirm initial receipt from Cashier.
2. **Data Entry:** Use STAC/RGN; write **N/A** in non-applicable fields.
3. **Authentication:** Stamp with Mark’s signature + Crimp/Seal.
4. **Digital Delivery:** If emailing, **encrypt the PDF** (do not use embosser for digital-only).
5. **Submission:** Student must email `admissions@sac.edu` with Name, SAC ID, DOB, Last 4 of SSN, and Receipt.

#### 2. Verification Forms (External Forms)
*Student provides a specific form for us to fill out.*
* ⚠️ **Restriction:** Cannot process until **Week 3** or after the **Refund/Drop Deadline**.
1. **Timing Check:** Verify the drop date has passed.
2. **Completion:** Complete form; attach a standard SAC letter if a deadline is urgent.
3. **Internal Routing:** Send to other departments via interoffice mail if required.
4. **Authentication:** Stamp with Mark’s signature + Crimp.

#### 3. Investigative / 3rd Party Requests
*Government agencies, employers, or clearinghouses.*
* **Investigative:** No fee. Attach unofficial transcript (stamped on back), seal in envelope, and mail.
* **3rd Party:** Refer to [National Student Clearinghouse](https://www.studentclearinghouse.org).
* **Written Requests:** Require a $2 check payable to **RSCCD**.
* **Phone Requests:** 🚫 **Prohibited.** Refer all callers to the Clearinghouse.

---

### ⚠️ Critical Compliance Notes

*   **FERPA:** If the request exceeds "Directory Information," a **Release Authorization Waiver** is mandatory.
*   **Non-Enrollment:** Only issue if the student dropped **before** courses appeared on the transcript. Otherwise, they must order an Official Transcript.
*   **SCC Students:** If they only attended Santiago Canyon College, issue a letter clarifying both are part of the same district but the student only took SCC courses.
*   **Special Cases:** Minor edits to letters require **Dean/Associate Dean** approval.
