## ID Requirements

ID is required for the following:

- Student Change of Information Request Form
- Transcript request form
- Permission to release education records
- Manually downloading applications from CCC Apply

!!! note
    ID is a common document to establish residency, but is **not required** if sufficient other documentation is submitted.
