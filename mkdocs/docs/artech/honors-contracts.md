## Processing Honors Contracts

Honors forms are processed **after all grades are posted**.

1. **Check:** Name, Student ID, Course, and Section for completeness.

2. **STAC** → Click on Course → **STNC** → Enter Comments/Printed Comments:
    - **Comments:** *Completed Honors Contract on file* + timestamp
    - **Printed Comments:** *Honors Contract Agreement Met*
    - **SAVE ALL**

3. **TRAN:** Verify changes appear on transcript.

4. **Email Kathy Patterson:** *"The SAC Honors form has been processed."*
   - If sent through Adobe Sign: sign online, then use the fully executed contract to file (will be resent via Adobe Sign). Forward to Secretary to save in instructor grade book.
