# Section Dates

## Missing Dates on SRGD

If section dates are missing from SRGD, verify the section was built correctly and contact ITS or the appropriate department to update.

---

## New Section

When building a new section, ensure all dates are populated correctly including:
- Start and end dates
- Add period dates
- Census date
- Last day to drop with W

---

## RG9010 Not Updating Dates

Run **RG9010** for the new term to push updated dates. If issues persist, contact ITS.

---

## Updating Add Periods for Sections Starting After Session Start

**BUSD** (or batch update process):
- Update add period start/end dates for sections that begin after the session start.
- Run for each affected term.

---

## Updating Section Dates (MIS Errors)

If section dates are causing MIS errors:
1. Identify the section and the date discrepancy from the MIS error report.
2. Correct dates in the section record.
3. Verify in CSXM after correction.

---

## Using BUSI to Modify Section Start and End Dates

**BUSI** allows bulk modification of section start and end dates. Use carefully — changes affect billing and registration eligibility.
