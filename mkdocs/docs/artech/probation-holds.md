# Probation Holds

## Probation Hold Rule Update

Probation hold rules in RLDE are updated as needed when probation status changes. Contact the Tech team or Hung for rule modification procedures.

---

## Probation Hold Release

When a student's probation/dismissal intervention is approved:

**PERC:**
1. Set end date to yesterday on the probation hold.
2. Detail into hold → timestamp → comment: *"Counselor submitted a probation/dismissal intervention form."*

**SREP:**
3. If the student had a TTPRO, TTPR2, or TTPR3: add Late Dates for the student.
