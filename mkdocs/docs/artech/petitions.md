# Petitions (AR Tech)

## California College Promise Grant (CCPG) Appeal

**STAC/STPE:** Review student's academic standing.

**PERC:**
1. If the student has completed and been approved for the CCPG appeal and meets criteria 1–3:
   - Update academic standing with the `AP` suffix (e.g., `A1AP`, `A2AP`).

## Late Add Petition

When a student needs to add a course after the add deadline:

1. Verify the Division Dean or instructor has approved the late add.
2. **RGSD:** Backdate to the add date as appropriate.
3. **RGN:** Register the student.
4. **RGSD:** Return to current date.
5. **STAC:** Add comment with timestamp documenting the late add and who approved it.

---

## P/NP After Grades

For students requesting P/NP conversion after grades have been posted:

1. Verify the request falls within the allowed timeframe and policy.
2. **STAC:** Update the grade.
3. Add comment with timestamp.
4. See also [Grade Changes — Change P/NP to Letter Grade](grade-changes.md#change-pnp-to-letter-grade) for the reverse process.
