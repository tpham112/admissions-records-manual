# Reinstatements (AR Tech)

Reinstatements at the AR Tech level follow the same process as [ARIII Instructor Change Forms — Reinstatement](../ariii/instructor-change-forms.md#reinstatement).

Key screens: **RGCS**, **STAC**, **RGN**.

The main steps:
1. RGCS — Reset status to A, set date to original drop date, Status Reason RE, delete drop grade and last attended date.
2. STAC — Adjust add time to be at least 1 minute after the drop time, delete any grade.
3. RGN — Verify the student is now registered.
